# My Class 11 Python Favorites 🐍

Welcome! This repository is a collection of some of my favorite Python programs from school. 

Instead of a complete archive, I’ve hand-picked a selection of **Class 11 scripts** that really helped me wrap my head around how **loops and conditions** work in programming.

### 💡 What I practiced here:
These scripts cover the basics that every beginner tackles. You'll find logic for:
* **Conditional Logic:** Checking for Leap Years, Odd/Even numbers, and Vowel/Consonant identification.
* **Number Crunching:** Identifying Prime, Armstrong, and Palindrome numbers.
* **Basic Math:** Finding the greatest/smallest numbers and calculating Simple Interest.
